# Sasi Vivek Neelapala — Portfolio

A modern, dark-themed personal portfolio website for **Sasi Vivek Neelapala** — AI & ML Engineer & Python Developer.

## ✨ Features

- 🎨 Premium dark theme with glassmorphism, gradient effects & micro-animations
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🤖 Showcases AI/ML projects, certifications, and professional experience
- 🌐 Multilingual resume support (English & Japanese 履歴書)
- 📝 Published articles section
- 🎓 Education & Certifications with clickable verification links
- 🌍 Language proficiency showcase (Japanese, English, Telugu, Hindi)
- 🧠 Community & Leadership activities
- ⚡ Built with React + Vite + Tailwind CSS + Framer Motion

## 🛠️ Tech Stack

| Layer     | Technologies                                      |
|-----------|----------------------------------------------------|
| Frontend  | React, TypeScript, Tailwind CSS, Framer Motion     |
| Build     | Vite                                                |
| Icons     | Lucide React                                        |
| Fonts     | Inter, Space Grotesk (Google Fonts)                 |

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at **http://localhost:8080/**
